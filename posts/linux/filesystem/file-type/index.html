Linux 文件类型

发表于 2018-08-30 | 分类于 notes-doc
字数统计: 147

Linux 文件属性第1位的标志，标识文件类型：- d l b c s p。

普通文件

-

目录

d directory

链接

l link

设备

与系统外设及储存等相关的文件，通常集中在 /dev 目录，一般分为两种：

b block
块设备文件，储存数据，如软硬光盘等，提供访问接口。

c character
字符设备文件，串行端口设备，如TTY(TeleTYpe)、PTY(Pseudo-TTY)、键鼠等。

套接口

s sockets
通信接口文件。

管道

p pipe
解决多个程序同时访问同一文件造成的错误问题。
