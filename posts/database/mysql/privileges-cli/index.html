MySQL 权限管理命令

发表于 2018-06-09 | 分类于 notes-cli
字数统计: 192

管理MySQL用户权限，控制用户访问库database、表table、执行权限、访问控制、授权等的权限等。

授权

'privileges'：用户的操作权限，如SELECT、INSERT、UPDATE、ALTER等，所的权限则为ALL
ON：表示对哪些数据库和表生效
'databasename'：数据库名，’‘表示所有数据库
'tablename'：表名，’‘表示所有表
TO：将权限授予哪个用户
WITH GRANT OPTION：表示允许用户将自己的权限授权给其它用户

刷新权限

对用户做了权限变更后，重新加载权限，将权限信息从内存写入数据库。
