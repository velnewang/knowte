Java 发行版本解释

发表于 2018-06-15 | 分类于 notes-doc
字数统计: 279

关于Java发行版本release的说明，包括版本version的长期支持版本long-term-support (LTS)和快速版本rapid及补丁patch的关键补丁更新版本Critical Patch Update (CPU)和补丁集更新Patch Set Update (PSU)。

Version

Java版本目前分为两种,长期支持版本long-term-support (LTS)和快速版本rapid。

Patch

从Java版本”7u71”开始，每次更新会同时发布两个版本。

奇数版本为关键补丁更新版本Critical Patch Update (CPU)，包含安全更新和关键漏洞修正，官方强烈建议”Oracle strongly recommends”更新并使用这个版本。

偶数版本为补丁集更新Patch Set Update (PSU)，包含了奇数版本的所有补丁，以及其他的非关键更新。由于此版本基本内容会包含在下一版关键跟新版本中，所以开发者或组织提前测试适配。

来自Oracle的说明cpu-psu-explained。

…
Starting with the release of Java SE 7 Update 71 (Java SE 7u71) in October 2014, Oracle will release a Critical Patch Update (CPU) at the same time as a corresponding Patch Set Update (PSU) for Java SE 7.

Java SE CPU releases are odd numbered versions
